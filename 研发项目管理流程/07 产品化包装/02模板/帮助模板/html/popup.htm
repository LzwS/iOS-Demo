弹出窗口的使用

功能

在各种联机帮助中，经常能看到弹出窗口。弹出窗口，顾名思义，即在当前的窗口中弹出一个新窗口，一般弹出窗口都比较小，主要用于名词解释、公告、新闻、还有广告。

在帮助模板中，提供了弹出窗口功能，该功能通过编写脚本来实现。下面介绍怎样制作一个弹出窗口。

操作步骤

  1. 编写脚本函数，主要代码如下，具体可以参见popup.js；
    function popup(str){
          if(sh = new ActiveXObject("Internet.HHCtrl")) {
                 sh.TextPopup(str,"宋体,10",9,9,0x800080,0xFFFFFF);
                 sh = "";
         }
    }
  2. 在需要使用弹出窗口的html文件的“<head></head>”中调用该脚本函数：
    <head>
    <script language=javascript src="../script/popup.js"></script>
    </head>
  3. 在需要设计弹出窗口的部分中加入弹出窗口中的内容即可：
    <body>
    <a href="Javascript:popup('弹出窗口中的内容')">弹出窗口</a>
    </body>

说明

代码中的紫色字体部分，表示自定义部分，即帮助制作者可以根据需要自行修改。
