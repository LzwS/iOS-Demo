翻页按钮的使用

功能

翻页按钮用来控制页面的跳转，一般包括上一级、下一级、上一页和下一页。在联机帮助中提供翻页按钮，可以方便帮助使用者快速跳转页面进行查看，而不需要再去目录结构中单击选择。下面提供翻页按钮的制作方法。

操作步骤

  1. 编写脚本函数，主要代码如下，具体可以参见page.js；

    2. function makevisible(cur,which){
        if (which==1)
            cur.filters.alpha.opacity=100;
        else
            cur.filters.alpha.opacity=50;
    }

    function MM_reloadPage(init) {
        if (init==true) with (navigator)  {
            if ((appName=="Netscape")&&(parseInt(appVersion)==4)) {
                document.MM_pgW=innerWidth; document.MM_pgH=innerHeight; onresize=MM_reloadPage; }
            }
            else if (innerWidth!=document.MM_pgW || innerHeight!=document.MM_pgH)
                location.reload();
    }
  2. 在需要使用翻页按钮的html文件的“<head></head>”中调用该脚本函数：
    <head>
    <script language="JavaScript" fptype="MM_reloadPage(true)"src="../script/page.js"></script>
    </head>
  3. 在body中加入以下代码即可：
    <body>
    <div id="PageBtn">
    <!-- 上一级链接 -->
    <a href="SysIntro.htm">
    <img src="../images/up.gif" style="filter:alpha(opacity=100)" onMouseOver="makevisible(this,0)" onMouseOut="makevisible(this,1)"></a>
    <!-- 上一页链接 -->
    <a href="SysIntro.htm"> <img src="../images/prev.gif" style="filter:alpha(opacity=100)" onMouseOver="makevisible(this,0)" onMouseOut="makevisible(this,1)"></a>
    <!-- 下一页链接 -->
    <a href="support.htm"> <img src="../images/next.gif" style="filter:alpha(opacity=100)" onMouseOver="makevisible(this,0)" onMouseOut="makevisible(this,1)"></a>
    <!-- 下一级链接 -->
    <a href="SS.htm"> <img src="../images/down.gif" style="filter:alpha(opacity=100)" onMouseOver="makevisible(this,0)" onMouseOut="makevisible(this,1)"></a>
    </div>
    </body>

说明

  1. 根据需要选取翻页按钮，不一定四个按钮都得用上；
  2. 根据需要修改按钮图片和按钮位置；
  3. 代码中的紫色字体部分，表示自定义部分，即帮助制作者可以根据需要自行修改。

示例
