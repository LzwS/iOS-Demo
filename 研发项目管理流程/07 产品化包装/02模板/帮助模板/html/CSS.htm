帮助模板

CSS层叠样式表单

功能

不同的网页会有不同的样式和风格，为了统一公司产品帮助的样式和风格，也为了能让帮助制作者快速制作出一份帮助文档，我们提供了一个通用的CSS样式表单：help.css，供大家在制作帮助时使用。

选择符 作用 样式 在html中使用方法 用途
body, table body和table的整体风格 定义body和table的字体为verdana, tahoma, arial或者宋体；字号：12px（象素）；字体颜色：黑色；文字装饰为：无；无外延边距 直接使用html标签即可，<body> </body>、<table> </table> 统一body、table的风格，每一个主题页面都使用
#head 页面顶部左侧 背景颜色：#005B7F；字体颜色：#FFFFFF（白色）；字号：14px；字体为粗体；高度：20px；文本位置：居中；垂直位置：居中；内 补丁边距 （上、右、下、左）：8px 25px 3px 20px；空格处理方式为：强制在同一行内显示所有文本，直到文本结束或者遭遇br对象 id="head" 定义页面顶部左侧样式
#headbg 页面顶部右侧 背景颜色#A4CBDB；宽度：100% id="headbg" 定义页面顶部右侧样式
#foot 页面底部 文本位置：居中；字体颜色：#999999（灰色）；字号：10px；
内补丁：0px 10px 0px 10px；页面位置：相对；上边框：1px solid（固态） #CCCCCC；上部外延边距：50px；高度：30px		 id="foot" 定义页面底部，一般为“版权所有”字样
#content 正文部分 外延边距（上、右、下、左）：0px 20px 0px 20px；行高：22px id="content" 定义正文部分，即夹在页面顶部和底部中间部分的样式
#content p 正文中的段落 行高：20px；外延上边距: 1em（相对长度单位）；外延底部边距：1em 直接在正文部分写<p></p>即可 定义正文中的段落的样式
h1 一级标题 字号：15px；外延边距（上、右、下、左）：40px 0px 20px 0px；字体颜色：#000080；下边框：1px solid #99999999 <h1></h1> 用于功能项名称，如本页面中的”CSS层叠样式表单”及下划线
h2 二级标题 字号：12px；外延上边距：1.2em；外延底边距：0.5em；字体加粗 <h2></h2> 用于一级标题下，如功能介绍页面中的：功能、操作步骤和说明
h3 三级标题 字号：12px；外延上边距：1.2em；字体加粗 <h3></h3> 用于二级标题下，比较少用
h4 四级标题 字号：12px；外延上边距：0.5em；字体加粗 <h4></h4> 比较少用
dl 列表 外延左边距：3em；外延底边距：1em <dl>
  <dt> </dt>
</dl>		 用于主题链接
dt 字体加粗；外延底边距：2em
ol 有序列表编号 外延顶边距：0.5em；外延左边距：2em；编号样式：阿拉伯数字 <ol>
   <li> </li>
</ol>		 用于描述操作步骤，或者说明部分
li 外延底边距：0.5em，行高：1.5em
ul 无序列表编号 外延顶边距：0.5em；外延左边距：1.5em <ul>
   <li> </li>
</ul>		 用于罗列内容
ol ul 有序和无序混和列表编号 外延顶边距：0.5em；外延左边距：1.5em；无序编号样式为实心圆 <ol>
   <li></li>
     <ul>
        <li></li>
     </ul>
   <li></li>
</ol>		 用于操作步骤或者说明中有序列表编号下还存在罗列内容的情况
table.Items 表格内容 外延边距：20px 0px 10px 0px；行高：130%（表示1.3倍行距） <table class="Items"> 定义正文部分的表格
.gray 灰色背景 背景颜色：#F1F1F1（灰色）；高：20px class="gray" 定义表格单元格的背景颜色和高度
caption 表格标题 行高：250%（2.5倍行距） <caption></caption> 用于表格标题
a:visited.caseindex 已经访问过的超链接 颜色：#0000FF；文字无装饰（即没有下划线） <a></a>或者<a class=caseindex></a> 用于超链接，比如主题链接，外部网页链接，或者邮箱链接
a:link.caseindex 未访问超链接 颜色：blue；文字无装饰（即没有下划线）
a:active.caseindex 被激活的超链接 颜色：blue；文字无装饰（即没有下划线）
a:.caseindex 超链接 鼠标定位时，超链接颜色为blue；鼠标离开时，超链接颜色为black
.pop 文本装饰 文本装饰为没有，即没有下划线或者闪烁等 class="pop" 用于去掉需要弹出窗口的文字的下划线
#PageBtn 翻页按钮位置 位置：绝对位置；右边距：25px；顶边距：8px id="PageBtn" 用于主题页面中的翻页按钮
#PageBtn img 每个翻页按钮图标 边框为0（无边框）；外延边距：0px 4px 0px 4px 直接使用<img src="图标位置">
