显示隐藏的使用

功能

由于页面空间的不足，或者内容的重要级别较低，可以将该内容隐藏起来，当需要显示的时候再显示。

操作步骤

  1. 编写脚本函数，主要代码如下，具体可以参见display.js；

    2. function initial() {
        var a;
        var i=initial.arguments.length;
        while(i>0) {
            a=initial.arguments[(i--)-1];
            if (a.style.display=="none")
                a.style.display="block";
            else
                a.style.display="none";
        }
    }
    function IsBlock(object1,object2) {
        if(object2.style.display=="block") {
           object1.src="../images/close.gif";
          object2.style.display="none";
       }
    else{
           object1.src="../images/open.gif";
           object2.style.display="block";
          }
    }
  2. 在需要使用显示隐藏的html文件的“<head></head>”中调用该脚本函数：
    <head>
    <script language=javascript fptype="initial()" src="../script/display.js"></script>
    </head>
  3. 在html文件的body中使用显示隐藏功能：
    <body  onload="initial(divxj)">
    <p><IMG id=imgxj src="../images/close.gif" width="14" height="10"><IMG src="../images/open.gif" border=0 width=0 height=0 style="display:none">
    <a title=显示隐藏 onmouseover="this.style.color='black'" onmouseout="this.style.color='blue'" href="javascript:IsBlock(imgxj,divxj)"><b>显示隐藏</b></a> </p>
    <ul id="divxj">
        <li>显示</li>
        <li>隐藏</li>
    </ul>
    </body>

说明

  1. 代码中的紫色字体部分，表示自定义部分，即帮助制作者可以根据需要自行修改；
  2. 代码中的灰色字体部分，用于解决html页面调用了javascript 函数文件，而该js文件中引用了图片，但是使用Html Help Workshop编译成.chm文件后，图片不能显示的问题。
